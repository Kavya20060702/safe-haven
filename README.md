# 🌿 Safe Haven

**Safe Haven** is an AI-powered mental wellness companion designed specifically for B.Tech students facing academic pressure, anxiety, stress, and loneliness. Through emotionally intelligent AI companions, voice interactions, and wellness tools, Safe Haven provides a safe and supportive digital space where students can express themselves and receive guidance.

---

## ✨ Features

* 🤖 **AI Companion System**

  * Choose from multiple unique companions with different personalities.
  * Each companion responds empathetically and adapts to conversations.

* 💬 **Real-Time Chat Support**

  * Natural conversations powered by AI.
  * Personalized responses based on user emotions and context.

* 🎙️ **Voice Interaction**

  * Talk directly with companions using speech recognition.
  * Text-to-speech responses for a more engaging experience.

* 😊 **Emotion-Aware Companions**

  * Animated characters display expressions based on conversation mood.
  * Dynamic reactions create a more human-like experience.

* 🚨 **Crisis Detection & Emergency Support**

  * Detects potentially harmful conversations.
  * Can alert trusted emergency contacts when necessary.

* 🌸 **Calming User Experience**

  * Lavender-themed interface designed to reduce anxiety.
  * Smooth animations and wellness-focused design principles.

* 🎥 **Immersive Welcome Experience**

  * Full-screen animated introduction before entering the platform.
  * Interactive companion selection system.

## 🎯 Problem Statement

Many B.Tech students experience:

* Academic stress
* Examination pressure
* Anxiety and burnout
* Social isolation
* Difficulty seeking emotional support

Safe Haven aims to provide accessible, judgment-free emotional assistance through AI-powered companions available anytime.

## 🚀 Getting Started

### Prerequisites

* Node.js
* npm

### Installation

```bash
git clone https://github.com/your-username/Safe-Haven.git
cd Safe-Haven
npm install
npm install -D @types/react @types/react-dom
```

### Environment Variables

Create a `.env.local` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
APP_URL=http://localhost:3000
```

### Run Development Server

```bash
npm run dev
```
Open:

```text
http://localhost:3000
```
